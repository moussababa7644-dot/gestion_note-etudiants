# Python – Bases
## Gestion de notes d’étudiants

Ce projet est un **gestionnaire de notes d’étudiants en console**, réalisé en Python.
Il permet d’ajouter des étudiants, d’enregistrer leurs notes, de calculer la moyenne
et de conserver les données grâce à un fichier JSON.

---

## Objectifs pédagogiques

Ce projet m’a permis de pratiquer :

- La manipulation des variables
- Les conditions (`if / elif / else`)
- Les boucles (`while`)
- Les fonctions
- Les dictionnaires
- La gestion des erreurs (`try / except`)
- La lecture et l’écriture de fichiers (`JSON`)

---

## ⚙️ Fonctionnalités

- Ajouter un étudiant
- Ajouter une note à un étudiant existant
- Afficher la moyenne d’un étudiant
- Sauvegarder automatiquement les données
- Recharger les données au démarrage du programme
- Quitter le programme proprement
