# Python – Bases (Jour 2)

Ce projet est un petit **gestionnaire de notes d’étudiants** en console.
Il permet d’ajouter des étudiants, d’ajouter des notes et de calculer la moyenne.

## Fonctionnalités
- Ajouter un étudiant
- Ajouter une note à un étudiant
- Afficher la moyenne d’un étudiant
- Quitter le programme

## Fichiers
- `gestion_notes.py` : programme principal

## Comment exécuter
```bash
python gestion_notes.py
